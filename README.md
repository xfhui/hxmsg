## 浣熊面试官｜小型互动网页

**浣熊面试官**是一个关于面试、表演与自我包装的互动网页作品。灵感来自我近期参加面试的经历：人们常常需要把零散的经历包装成完整故事，把不相关的背景解释成岗位优势，把很小的项目讲得足够宏大。

作品将这种现实中的荒诞转化为一场可爱的模拟面试。用户面对一只会提问、会追问、会吐槽的浣熊面试官，并根据不同的荒唐岗位编造答案。越不合理的职位，越夸张的回答，反而越接近现实面试中半真半假的表演状态。

技术上，项目使用 HTML、CSS 和 JavaScript 构建，并接入 **GLM-4-AirX 模型**，使浣熊面试官能够根据用户的回答进行实时交互、针对性追问与反馈。结合角色素材、拟声音效和颜文字反馈，作品在轻量网页中创造出一个可爱但带有讽刺意味的 AI 面试场景。

在这只浣熊的提问背后，作品试图追问：我们到底是在展示真实的自己，还是在扮演一个更适合被录用的人？
